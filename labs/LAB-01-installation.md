**LAB-01 - Installation de Suricata**



**Objectif:** 

Installer et préparer un environnement Suricata fonctionnel sous Windows.



**Environnement:**

Système : Windows

IDS : Suricata 8.0.5

Capture réseau : Npcap

Terminal : PowerShell



**Étapes réalisées:**

1\. Installation de Suricata

Suricata a été installé dans : C:\\Suricata



Vérification de l'installation : .\\suricata.exe --build-info



**Résultat attendu :**

Version de Suricata affichée

Support Npcap actif



2\. Installation de Npcap

Npcap permet à Suricata d'accéder aux interfaces réseau Windows afin d'analyser les paquets.



**Résultat**

Installation réussie.

Suricata est prêt pour la configuration réseau.
